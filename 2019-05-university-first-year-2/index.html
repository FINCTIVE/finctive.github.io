西安-夏-大一下

FINCTIVE 2019-05

总感觉时间被浪费掉了！我应该能再多做一些有意思、有意义的事情！今天是即刻 APP 关闭服务器的第二天，我终于想起了这个挂在 Github 上的博客……

我在这个学期花了大量时间来写 Unity3D 引擎游戏脚本，其次自学的内容还有音乐制作和矢量绘图基础。说实话感觉一般，总觉得自己能做更多的事情。自律太难了😭，划水真有趣。（加之学校带来的多重大山，学习进展缓慢似乎是必然的。）

不过进步还是有的，因为看开了许多事情，相比上学期我挤出了更多的时间自学（于是成绩好了一些）。我不好意思说我是怎么挤时间的（嗯！）希望下学期的我能更自律吧，在清单上把计划一行一行的计划完成，感觉太好了。

I have never let my schooling interfere with my education. —— Mark Twain

本学期的最大成就：

  • 开始开发一个开源的游戏项目 FINCTIVE/lost-in-the-wilderness-nightmare 这是一个练手项目，我打算边学 Unity 边给它加入新的功能。
  • 参加了CiGA Gamejam 2019 (西安站) ！我的队伍是两人小队，都是程序员，现场没有找到美术、策划和音效队友。于是，我挂着程序的牌子干了美术、策划和音效的工作😂 最终游戏没有做完，太惨了，我在Gamejam开了一个大坑，然后乖乖送人头。不过在路演环节，大家对我们的点子反响不错！那天受到的关注也比较多，还认识了一些新的朋友。上大学以来第一次感觉做游戏好棒，长时间以来的学习与练习得到了外界的肯定。有志开发游戏的学生朋友们，强烈建议参加一次Gamejam！

摸鱼活动？这个学期最大的时间黑洞是《Apex英雄》，这款游戏最吸引我的亮点有：

  • 优秀的交流系统，上手之后，队友间几乎可以不开语音进行交流。
  • 大部分情况下，战术与队友间的合作比单人实力更重要。相对其他FPS游戏来说，《Apex英雄》中玩家的血量更厚，跑的更快，跳的更高，更难打中。加之各类技能的作用，逃命的几率大大增加。在《彩虹六号：围攻》中，一人打倒三人的情况很有可能发生。而在《Apex英雄》中，几乎不可能（除非逐个消灭）。
  • 队友互相帮助的机会变多了，比如：玩家“成盒”后，队友仍能在重生点复活玩家。
  • 大逃杀类游戏有趣的随机性，还有收集装备逐渐变强的快感。

创造新的游戏玩法，感觉像在创造一个新的世界。
